# iTrellisProj v1.0.0

This is a Windows Desktop application that will calculate how to settle group expenses. This application will assume that all expenses are split evenly.

Simply double click the run.bat file to start the application.

Usage:

1. Adjust names as required, and blank names will be interpreted as nobody. If there are 2 name entries and a blank name, all expenses will be split amongst the 2 name entries.

2. After adding your expense, click the "Update Expense" button to 
	a. Calculate everyone's total expenses
	b. Calculate what everyone is owed
	c. Have the application prescribe payments to settle the 	debt amonst your party

3. Click the "Add Expense" button to add another expense to the selected person's expense list. This will automatically cause the application to update everyone's latest expenses and settlements.
